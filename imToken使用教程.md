# imToken 钱包使用教程

> 下面给你 imToken 专属、一步一图级文字版 授权与取消授权完整教程，照着做就不会踩坑。

---

## 一、imToken 连接 DApp（登录，只读）

1. 打开 imToken → 底部点 **浏览**

2. 输入 DApp 网址（或从推荐进入）

3. 点页面上 **Connect Wallet / 连接钱包**

4. 选择 **imToken** → 钱包弹出：**请求连接**
   - 只看：**DApp 名称 + 你的地址**

5. 点 **确认** → 连接成功（仅登录，不能转钱）

---

## 二、imToken 合约授权（Approve，能转钱，高危）

1. 在 DApp 操作（Swap / 挖矿 / 质押等）→ 点 **授权 / Approve**

2. imToken 弹出交易，**必须检查 3 点**：

   - **合约地址**：是否是项目官方合约（可在区块链浏览器核对）
   
   - **授权额度**：尽量选 **自定义小额**，**不要无限授权**
   
   - **Gas 费**：是否合理

3. 确认无误 → 输入钱包密码 / 指纹 → 支付 Gas

4. 上链成功 → 该合约获得你该代币的**转账权限**

---

## 三、场景一：代币授权（给其他钱包别转 TRX/TRC20 权限）

### 准备

- 打开 imToken → 进入 **TRON（TRX）钱包**

- 钱包内有 **≥30 TRX**（支付 Gas / 能量）

- 准备好**目标钱包地址**（要授权的钱包地址）

### 操作步骤（通过 TRONSCAN 授权）

1. 进入 TRX 钱包 → 首页功能栏**左滑** → 点击 **「授权管理」** → 自动跳转到 **TRONSCAN** 并连接钱包

2. 切换语言：右上角菜单 → **Preferences** → 简体中文 → Save

3. 找到 **「授权」/ 「Approve」** 入口（或直接访问 TRONSCAN 授权页面）

4. 填写关键信息：

   - **代币**：选择 TRX（或你要授权的 TRC20，如 USDT-TRC20）
   
   - **被授权地址（Spender）**：粘贴**目标钱包地址**（要授权的钱包）
   
   - **授权额度**：拒绝无限额度，改为**实际需要的最小金额**（如 100 TRX）

5. 确认交易：

   - 核对地址、代币、额度无误
   
   - 输入钱包密码 / 指纹 → 支付 Gas → 确认上链

6. 完成：交易成功后，目标钱包即可在额度内划转你的代币

### 如何查看 / 撤销授权

1. 回到 TRX 钱包 → **「授权管理」** → TRONSCAN

2. 进入 **「授权列表」** → 查看所有授权记录

3. 撤销：找到记录 → 点击 **「取消授权」** → 确认交易

## 四、imToken 钱包 TRX 代币授权给其他钱包后转账还需要输入密码吗？

### 分两种情况，核心区别是**谁发起转账**：

### 一、你自己在 imToken 里转账（用你的钱包）

- **需要输入密码**。

- 你给其他钱包做的是**代币授权（Approve）**，只是允许对方在额度内代你划转，**不影响你自己正常转账**。

- 你自己发起的每一笔转账，依然要走 imToken 的安全流程：**输入密码 / 指纹 / 面容 → 签名 → 上链**。

### 二、被授权的那个钱包发起转账（用对方的钱包）

- **你不需要输入密码**。

- 对方在他自己的钱包里操作，**用他自己的私钥签名**，直接把你授权额度内的 TRX/TRC20 转走。

- 全程**不需要你确认、不需要你输密码**。

---

## 三、关键安全提醒（非常重要）

### ⚠️ 授权 = 给对方"代扣权"

- 一旦授权，对方可以在额度内**随时划走你的币**，无需你同意。

- **永远不要授权给不信任的地址**。

### ⚠️ 无限授权 = 永久风险

- 很多 DApp 默认请求"无限额度"，方便但危险。

- **强烈建议**：每次只授权**实际需要的小额**，用完就撤销。

### ⚠️ 定期检查授权列表

- 进入 imToken → TRX 钱包 → **授权管理** → TRONSCAN

- 查看所有授权记录，**撤销不再使用的授权**。

### ⚠️ 转账密码只保护"你自己发起的操作"

- 密码无法阻止**被授权方**的划转行为。

- 安全的根本是：**谨慎授权 + 小额授权 + 及时撤销**。

---

## 总结

| 场景 | 是否需要密码 | 谁在操作 | 风险 |
|------|------------|---------|------|
| **你自己转账** | ✅ 需要 | 你在 imToken 里发起 | 低（正常流程） |
| **被授权方转账** | ❌ 不需要 | 对方在他的钱包里操作 | 高（对方可随时划走） |

**最佳实践**：
1. 只授权给**官方合约**或**绝对信任的地址**
2. 授权额度设为**最小必要值**
3. 用完立即**撤销授权**
4. 定期检查**授权列表**，清理无用授权

## 常见问题

### Q1: 如何查看我授权了哪些地址？
A: imToken → TRX 钱包 → 授权管理 → TRONSCAN → 授权列表

### Q2: 授权后对方可以转走我所有的币吗？
A: 只能转走你授权的代币类型和额度内的数量，其他币种不受影响

### Q3: 撤销授权需要 Gas 费吗？
A: 是的，撤销授权也是一笔链上交易，需要支付少量 TRX 作为 Gas 费

### Q4: 无限授权有什么风险？
A: 如果授权的合约被黑客攻击或是钓鱼合约，你的所有该代币都可能被盗

### Q5: 连接钱包和授权有什么区别？
A: 连接钱包只是让 DApp 看到你的地址（只读），授权则是给予转账权限（可写）
